# Templates

Reusable templates for common AI/ML project documentation tasks.

## Contents

- [Experiment Log](./experiment-log.md) - Track ML experiments
- [Model Card](./model-card.md) - Document models
- [Dataset Card](./dataset-card.md) - Document datasets

## Purpose

Templates provide standardized formats for:
- **Consistency** - Uniform documentation across projects
- **Completeness** - Ensure nothing is missed
- **Efficiency** - Faster documentation creation
- **Quality** - Proven structure for good documentation
- **Searchability** - Consistent structure aids discovery

## Using Templates

1. Copy the template
2. Fill in all sections
3. Customize for your project
4. Get reviewed
5. Publish

## Best Practices

- Don't skip sections - fill in N/A if not applicable
- Keep updated as project evolves
- Version your templates
- Get feedback from users
- Iterate and improve

# Overview Phase

The foundation of every successful AI/ML project begins with clear definition and alignment across all stakeholders.

## Contents

- [Project Charter](./project-charter.md) - Define project scope, objectives, and authority
- [Stakeholders](./stakeholders.md) - Identify and manage all stakeholders
- [Success Criteria](./success-criteria.md) - Define measurable success metrics

## Purpose

The Overview phase establishes:
- **Clear objectives** - What the project aims to achieve
- **Stakeholder alignment** - Who is involved and what are their expectations
- **Success definition** - How will we measure if the project succeeds

## Key Deliverables

1. Project charter document
2. Stakeholder register
3. Success criteria and KPI definitions

## Search Tips

### Using GitHub Search
```
# Search for specific technology
repo:abedhraiz/how_to kubernetes deployment

# Search in specific directory
repo:abedhraiz/how_to path:docs/infrastructure-devops/ terraform

# Search by file type
repo:abedhraiz/how_to extension:md prometheus

# Search for code examples
repo:abedhraiz/how_to language:yaml kubernetes
```

### Using Local Search
```bash
# Search all guides for a term
grep -r "docker-compose" docs/

# Find guides mentioning specific technology
find docs/ -name "*.md" -exec grep -l "PostgreSQL" {} \;

# Search within specific category
grep -r "monitoring" docs/infrastructure-devops/
```

### Finding Related Content
- Each category README includes **Related Categories** section
- Individual guides include cross-references at the bottom
- Check the **Technology Index** for complete alphabetical list
